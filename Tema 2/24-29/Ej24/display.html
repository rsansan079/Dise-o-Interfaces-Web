Ejemplo de la Propiedad CSS Display

La propiedad displayen CSS determina cómo se comporta un elemento en el flujo del documento.

  • block: El elemento ocupa todo el ancho disponible y comienza en una nueva línea.
  • inline: El elemento ocupa solo el espacio necesario y no interrumpe el flujo en línea.
  • inline-block: Combina características de block (permite definir ancho y alto) y inline.
  • none: Oculta el elemento completamente, eliminándolo del flujo del documento.
